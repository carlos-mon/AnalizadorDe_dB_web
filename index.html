Analizador de dB

Proyecto para Funciones y Matrices - INACAP

Prueba la App en Vivo

Esta aplicación mide decibeles en tiempo real, guarda historiales y alerta sobre niveles peligrosos para personas con TEA.

Características Técnicas:

  • Lenguaje: Kotlin (Nativo)
  • Matemáticas: Escala Logarítmica (dB) y cálculo RMS.
  • Base de Datos: Room (SQLite) para persistencia histórica.
  • Interfaz: Jetpack Compose con gráficos en tiempo real.
